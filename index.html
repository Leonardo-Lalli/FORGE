Tutto il necessario. Niente di superfluo.

Registra serie, ripetizioni e carichi. FORGE calcola il volume totale per gruppo muscolare e ti mostra i trend nel tempo.

// Self-Hosting

Il tuo server, le tue regole.

Avvia il backend FORGE in pochi secondi con Docker e PocketBase. Un singolo comando per sincronizzare i tuoi allenamenti tra dispositivi — completamente self-hosted.

  • ▸ Backend leggero basato su PocketBase (SQLite integrato)
  • ▸ Setup con un solo comando tramite Docker Compose
  • ▸ Sincronizzazione multi-dispositivo opzionale e crittografata
  • ▸ Zero costi ricorrenti, zero vendor lock-in
forge — backend setup 
# 1. Clona il repository ufficiale
$ git clone https://github.com/forge-app/backend.git
$ cd backend

# 2. Avvia PocketBase in background
$ docker compose up -d

[+] Running 2/2
 ✔ Network forge_default     Created
 ✔ Container forge-pocketbase  Started

# 3. Backend pronto su http://localhost:8090
$ docker compose ps
