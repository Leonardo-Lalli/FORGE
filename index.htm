Self-Hosting in 30 secondi

Avvia il backend sul tuo Raspberry Pi, NAS o server Proxmox. I tuoi dati restano dentro casa tua.

# 1. Clona la repo git clone https://github.com/Leonardo-Lalli/FORGE.git && cd FORGE # 2. Avvia PocketBase con Docker docker compose up -d # 3. Installa l'APK sul telefono # 4. Apri FORGE → Inserisci IP del server → Pronto!

Funziona su LAN. Per accesso remoto: DuckDNS + Nginx Proxy Manager (guida nella repo).
