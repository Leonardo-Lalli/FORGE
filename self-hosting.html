// Self-Hosting

Il tuo server, le tue regole.

FORGE è costruito attorno a un'idea semplice: i tuoi dati di allenamento dovrebbero appartenere solo a te. Con PocketBase self-hosted e Docker, avvii il backend in 30 secondi su qualsiasi macchina Linux.

// Come funziona

Tre passi per l'indipendenza

01

Avvia il backend

docker compose up -d sul tuo server. PocketBase parte con database, auth e storage in un container.

02

Installa l'app

Scarica l'APK, aprilo. Al primo avvio inserisci l'IP del tuo server (es. 192.168.1.50:8090).

03

Allenati in libertà

Registrati, inizia ad allenarti. I tuoi dati non lasciano mai la tua rete. Zero costi cloud, per sempre.

// Vantaggi

Perché self-hostare FORGE

Il tuo server, le tue regole

Nessun cloud di terze parti. PocketBase gira su un singolo binario con database SQLite, autenticazione e file storage integrati. Lo avvii dove vuoi: Raspberry Pi, NAS, mini-PC, Proxmox, VPS.

Privacy by design

I dati degli utenti non attraversano mai internet verso server sconosciuti. Tutto rimane dentro la rete locale. Nessun tracker, nessuna pubblicità, nessuna analytics di terze parti.

LAN-first, offline-ready

L'app comunica direttamente col server via Wi-Fi locale. Se internet cade, l'allenamento continua. I workout si salvano in SQLite sul telefono e si sincronizzano appena la rete torna.

Accesso remoto opzionale

Con DuckDNS + Nginx Proxy Manager puoi esporre il tuo server su internet con HTTPS (Let's Encrypt) e certificate pinning. I tuoi amici possono seguirti anche da fuori casa.

// Hardware consigliato

Cosa ti serve

🍓

Raspberry Pi 4/5

~60€. Silenzioso, consuma 5W. Perfetto per uso personale o familiare. Supporta 10-50 utenti attivi.

💻

Mini-PC / NUC

~200€. Più potente, ideale per palestre o community. Supporta 100-500 utenti. Puoi farci girare anche Nginx.

☁️

VPS (Hetzner, Linode)

~5€/mese. Server virtuale in datacenter. Accesso pubblico nativo. Ideale per community online.

// FORGE Local Business

Un modulo B2B per le palestre

L'architettura self-hosted permette a un proprietario di palestra di lanciare il backend su un mini-PC locale e creare un sistema gestionale e social privato per tutta la struttura.

Zero costi SaaS

Niente abbonamenti o licenze per postazione. Da 50 a 5.000 iscritti, il costo software è 0€/mese.

Community locale

Chi si allena nella stessa sala vede i PR degli altri in tempo reale, con classifiche interne della palestra.

Privacy e GDPR al 100%

I dati su salute e performance non lasciano mai l'edificio. Un vero punto di marketing.

Offline / LAN First

Se internet si interrompe, l'app continua a funzionare via Wi-Fi locale. Zero downtime durante gli allenamenti.

// Quick start

Avvia il backend ora

# 1. Clona la repo

git clone https://github.com/Leonardo-Lalli/FORGE.git

cd FORGE

# 2. Crea il file .env (opzionale per encryption key)

echo "PB_ENCRYPTION_KEY=your_secret_key" > .env

# 3. Avvia PocketBase con Docker Compose

docker compose up -d

# 4. Verifica che sia in esecuzione

docker compose logs

# 5. Crea un account admin (solo prima volta)

Apri http://IP-DEL-SERVER:8090/_/ nel browser

Crea il primo account admin

# 6. Configura l'app sul telefono

Apri FORGE → Setup iniziale → Inserisci http://IP-DEL-SERVER:8090

# Per trovare l'IP del server:

ip addr show | grep inet # Linux

ipconfig # Windows

ifconfig # macOS
